Project 1:  Windows RFIDler Driver Install
            Graphical interface to install driver .inf & .cat files
            for Aperture Labs RFIDler LF.

Project 2:	RFIDler Monitor
            Graphical monitor that lists which USB ports an RFIDler is 
            currently connected to, and watches for changes.
			Also watches for devices connected running Microchip HID Bootloader
			(potentially an RFIDler waiting to be programmed), and other 
			categories of serial port.
            Tool for Aperture Labs RFIDler LF.
